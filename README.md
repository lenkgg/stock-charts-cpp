# stock-charts-cpp
基于 C++/Qt 实现K线图和指标。
- 纯头文件实现，代码量极少，方便阅读和二次开发。
- 抽象 Painter 兼容任意 UI 框架，按需组装功能和界面。
- 提供 Qt Painter 实现，可用 Win/Mac 直接运行，支持鼠标键盘交互。
