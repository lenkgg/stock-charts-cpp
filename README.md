# stock-charts-cpp
基于 C++17 实现K线图（包含指标和绘图）

## 开发计划

### v0.0.1
- 创建 Qt6 项目（可替换其他UI库）
- Pure C++ 实现图表框架
- 实现基本的K线图架构

### v0.0.2
- K线图(candlestick chart, k-line chart)
- 完善 KChart 功能及交互

### v0.0.3
- 分时图(timeframe chart, timesharing chart)
- 完善 TChart 功能及交互

### v0.0.4
- 股票叠加

### v0.0.5
- 画图工具

### v0.0.6
- 完善指标及公式解析

### v0.1
- 发布首个稳定版本
