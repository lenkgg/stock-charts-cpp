# stock-charts-cpp
基于 C++/Qt 实现K线图和指标
- 纯头文件实现，代码量极少，方便阅读和使用。
- Qt 跨平台，可用 Win/Mac 直接运行。
- 抽象 Painter 兼容任意 UI 框架。
